Warfare (Node.js army builder game)

This is a simple army building- and fighting game. The goal was to have a game that could not only run a battle between two user armies, but kept track of each user's army between logins and also had a time based rebuilding mechanism.

On the server side, I make use of Express.js and lowdb. Within the database, it keeps track of the armies created as JSON objects including the chosen username, the number of three different types of units, the number of upgrade points available to the player, and the last login timestamp. To assign new upgrade points, at login it compares the current timestamp to the previous last login (before replacing it).

On the client side, the html is populated via JavaScript template strings. At login, the user's army is set to local storage so that they are logged in until local storage is cleared. To recruit units or choose another army to battle I make use of promises- a fetch is performed and acted upon via a .then() command.
