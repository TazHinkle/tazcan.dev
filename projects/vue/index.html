MongoDB/Vue App

The client side is a Vue template. It consists of an input, a button, and a series of divs which are generated at run time when the response from get request is sent from the database. The divs populate as green if no errors were encountered, and red otherwise.

The server side is Node.js working with HTTP client axios.js, connecting to a Mongo database. I ran a local instance of MongoDB in a Docker container and used the RoboMongo GUI to create the database and the collection.


Vue Stat Calculator App

In this app, I consume the fanzey pokemon API to reproduce some of the pokemon stats in a list. This list is sortable by each of the characteristics chosen. One characteristic, "Stat Sum," is calculated in a Vue computed. The template populates a table at runtime with the response from the API call.
