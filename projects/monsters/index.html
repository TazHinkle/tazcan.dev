On this page I consume the open5e API to reproduce details from the Monster Manual in a list, and then provide details in a chart. This project makes use of the Darkly Bootstrap CSS theme, and Chart.js library. You can sort by any of the displayed details and click on a monster's name to display a chart of their attributes.

Since the data is static, to be considerate to the API, I stored the data in a GitHub gist. As a consideration to users, a second layer of caching is used, storing data in local storage. I make use of GitHub Pages for the final product.

I implemented pagination and list sorting with JavaScript functions, JavaScript template strings for all of the page content rendering, and managed all application state in a simple url hash routing function.

A previous API consumption project made use of NASA's Picture of the Day API and is pictured (and linked to) on the bottom right.
